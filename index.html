EduBot eğitim asistanıdır. Gerçek yapay zekâ için Ayarlar bölümünden güvenli API endpoint ekleyin.

Yapay Zekâ Ayarları

GitHub Pages statik çalışır. API anahtarını doğrudan bu dosyaya yazmak güvenli değildir. En doğru kullanım: Vercel/Netlify/Cloudflare Worker gibi bir yerde kendi güvenli endpointinizi oluşturup aşağıya endpoint adresini yazmaktır. Endpoint şu JSON'u almalı: {system, message, history} ve {answer} döndürmelidir.
